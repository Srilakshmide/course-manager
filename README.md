# Course Manager Application

## Description
This is a minimal starter scaffold for a Course Manager application using:
- Next.js 16 (app router)
- React
- Tailwind CSS
- Node.js + Mongoose (MongoDB)

How to run (locally):
1. Install dependencies:
   ```bash
   npm install
   ```
2.  Run dev server:
   ```bash
   npm run dev
   ```

## Features
- `/app` — Next.js app routes and API route handlers
- `/models` — Mongoose models
- `/lib/mongodb.js` — MongoDB connection helper
- `/components` — React components

- User authentication (login/register)
- Course CRUD operations
- Slug-based routing
- Protected routes
- Responsive UI

## Tech Stack
- Next.js (App Router)
- MongoDB Atlas
- JWT
- Tailwind CSS
